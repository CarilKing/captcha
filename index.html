Simple CaptchaGo package captcha implements generation of image

GITHUB PAGE

Start using it

go get github.com/afocus/captcha

How to use

cap := captcha.New()
// Must set a font, Other settings have default values
cap.SetFont("comic.ttf")
img, str := cap.Create(4, captcha.NUM)
